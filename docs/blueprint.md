# **App Name**: NetGuard AI

## Core Features:

- KPI Dashboard: Display a dashboard with key performance indicators (KPIs) and alerts related to network monitoring.
- Command Execution: Allow users to input commands to be executed on remote network devices via SSH.
- Conditional Rules Engine: Implement a system to define rules based on conditional logic (QUANDO / SE / AÇÃO) for automated responses to network events.
- AI Rule Suggestions: Use a generative AI tool to suggest rules based on the analysis of historical network performance data and common ISP issues.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey reliability and trust.
- Background color: A dark, desaturated blue (#242B37) for a modern, tech-focused feel.
- Accent color: A contrasting yellow (#FFD700) for alerts, highlights, and important actions to draw attention.
- Use a clean and modern sans-serif font.
- Utilize minimalist, line-based icons for representing network devices, alerts, and actions.
- Employ a grid-based layout to organize dashboard elements and ensure a consistent and responsive design across different screen sizes.
